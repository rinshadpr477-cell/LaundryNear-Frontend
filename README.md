# LaundryNear 

A modern laundry service platform connecting customers with nearby laundry shops for pickup, washing, ironing, and doorstep delivery.

## Live Demo

Frontend: https://laundry-near-frontend.vercel.app

Backend: https://laundrynear-backend.onrender.com

## Features

- Customer Dashboard
- Shop Dashboard
- Delivery Dashboard
- Admin Dashboard
- Order Tracking
- Razorpay Payment Integration
- JWT Authentication
- Role-Based Access Control
- Laundry Shop Management
- Real-Time Order Status Tracking

## Tech Stack

### Frontend
- React
- Vite
- React Router
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- MongoDB Atlas
- JWT
- Razorpay
- Multer
